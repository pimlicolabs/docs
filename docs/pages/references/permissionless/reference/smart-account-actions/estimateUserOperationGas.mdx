# estimateUserOperationGas

Estimates the `preVerificationGas`, `verificationGasLimit`, and `callGasLimit` needed to validate and execute a user operation against the current chain state. `SmartAccountClient` wraps viem's `BundlerClient.estimateUserOperationGas`, so the [full list of options is documented in viem](https://viem.sh/account-abstraction/actions/bundler/estimateUserOperationGas).
