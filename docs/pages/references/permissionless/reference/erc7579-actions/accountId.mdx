# accountId

Returns the account ID of the smart account as defined in [ERC-7579](https://eips.ethereum.org/EIPS/eip-7579) — a string that identifies which ERC-7579 implementation the account follows so modules can be installed against the correct interface.

Use the `accountId` method to get the accountId of the smart account.

```ts
const accountId = await smartAccountClient.accountId()
console.log(accountId)
```

## Returns

- **Type:** `string`

The account id of the smart account.

## Parameters

### account (optional)

- **Type:** `SmartAccount`

If your `SmartAccountClient` doesn't have an account, you should provide one here.
