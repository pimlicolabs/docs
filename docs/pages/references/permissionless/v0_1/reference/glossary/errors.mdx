# Errors

_You are viewing permissionless.js v0.1 documentation. [View the latest documentation →](/references/permissionless)._ A glossary of every error class thrown by permissionless.js v0.1, grouped by the layer that raises them (EntryPoint, smart account, paymaster, etc.) so you can quickly find what went wrong.

All errors in permissionless.js extend the standard viem [`BaseError`](https://github.com/wevm/viem/blob/main/src/errors/base.ts).
