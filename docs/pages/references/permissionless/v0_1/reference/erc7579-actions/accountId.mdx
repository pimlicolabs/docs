# accountId

_You are viewing permissionless.js v0.1 documentation. [View the latest documentation →](/references/permissionless)._ Returns the account ID of the smart account as defined in [ERC-7579](https://eips.ethereum.org/EIPS/eip-7579) — a string that identifies which ERC-7579 implementation the account follows so modules can be installed against the correct interface.
