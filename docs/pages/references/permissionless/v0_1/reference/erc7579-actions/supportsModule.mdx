# supportsModule

_You are viewing permissionless.js v0.1 documentation. [View the latest documentation →](/references/permissionless)._ Returns `true` if the smart account supports a given [ERC-7579](https://eips.ethereum.org/EIPS/eip-7579) module type (validator, executor, hook, or fallback) — check this before attempting to install a module of that type.

## Returns

- **Type:** `boolean`

True if the module type is supported, false otherwise.

## Parameters

### type

- **Type:** `ModuleType`

Type of the module to check. Accepted values are `"validator" | "executor" | "fallback" | "hook"`.
