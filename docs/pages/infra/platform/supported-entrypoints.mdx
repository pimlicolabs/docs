## EntryPoint Versions

- **v0.6**: The original EntryPoint contract deployed for ERC-4337
- **v0.7**: Updated EntryPoint with improved security features and optimizations
- **v0.8**: Latest EntryPoint version with additional improvements

When interacting with the Pimlico API, you can specify which EntryPoint version to use based on your smart account implementation.
