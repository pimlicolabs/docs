# pimlico_getMagicSpendStakes

```json
{
    "jsonrpc": "2.0",
    "method": "pimlico_getMagicSpendStakes",
    "params": [
        "0x129a50835EB5EFA2253b2f0Ec2fBF84b2cEe1fF9",
        "0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE"
    ],
    "id": 1
}
```
