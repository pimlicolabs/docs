# pimlico_getMagicSpendStakes

This method returns the list of stakes for a given user and asset.

## Request

```json
{
    "jsonrpc": "2.0",
    "method": "pimlico_getMagicSpendStakes",
    "params": [
        "0x8C6bdb488F664EB98E12cB2671fE2389Cc227D33",
        "0x46Bc62B13e890f0B180e6E1A475bEC4Cc895aEA3"
    ],
    "id": 1
}
```
