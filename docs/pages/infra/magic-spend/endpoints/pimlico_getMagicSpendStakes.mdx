# pimlico_getMagicSpendStakes

:::warning[Warning]

The MagicSpend++ protocol is in beta. API is subject to change with short notice.

:::


This method returns the list of stakes for a given user.

## Request

```json
{
    "jsonrpc": "2.0",
    "method": "pimlico_getMagicSpendStakes",
    "params": [
        {
            "account": "0x77d1f68C3C924cFD4732e64E93AEBEA836797485"
        }
    ],
    "id": 1
}
```
