# pimlico_getMagicSpendContracts

:::warning[Warning]

The MagicSpend++ protocol is in beta. API is subject to change with short notice.

:::

This method returns the addresses for two MagicSpend++ contracts: `MagicSpendStakeManager` and `MagicSpendWithdrawalManager`.

## Request
```json
{
    "jsonrpc": "2.0",
    "method": "pimlico_getMagicSpendContracts",
    "params": [],
    "id": 1
}
```

## Response

```json
{
    "jsonrpc": "2.0",
    "id": 1,
    "result": {
        "stakeManagerAddress": "0x98f521f92C588B9690193932667c43153502f206",
        "withdrawalManagerAddress": "0x4d43b57AF83246cdBF36104Bc4CC1866d7944115"
    }
}
```
