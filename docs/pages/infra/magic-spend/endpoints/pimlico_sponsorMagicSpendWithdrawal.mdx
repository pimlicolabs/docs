# pimlico_sponsorMagicSpendWithdrawal

:::warning[Warning]

The MagicSpend++ protocol is in beta. API is subject to change with short notice.

:::

This method accepts the withdrawal parameteres along with a signature over this parameters. The signature should be done by the `operator` of the `Allowance` message.

```json
{
    "jsonrpc": "2.0",
    "method": "pimlico_sponsorMagicSpendWithdrawal",
    "params": [
        {
            "recipient": "0x100c817c465034CcbEC8Dc0436dbB6C2983032B2",
            "asset": "0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE",
            "amount": 123,
            "salt": 0,
            "signature": "0x606b713744cfab2a440c07a564c7630195f974372b3c1ef62f10048f84931fdd7ad89b9ea48212994960b1e09ab34ead8c0a7e05aeae89427d3200d8e4463a1e1c"
        }
    ],

}
```
